##Flow of Your Editor:
User presses key
    ↓
OS creates input event
    ↓
SDL receives event
    ↓
Character added to text buffer
    ↓
Screen cleared
    ↓
Each character processed:
        Character
            ↓
        Glyph loaded
            ↓
        Glyph rasterized
            ↓
        Bitmap created
            ↓
        Surface created
            ↓
        Texture created
            ↓
        Texture rendered
    ↓
Cursor rendered
    ↓
Frame presented on screen
    ↓
Temporary memory cleaned
    ↓
Loop repeats


About Space and Tab:
-> no visible bitmap
-> but still advances cursor
=> So rendering skipped, but cursor moves.
